## Cipher3000

### Description

Simple script which is ciphering the text files.

### Requirements
- Python 3.10

### How to run it?
1. Clone repository.
2. Go to cloned directory.
3. Run following commands
```bash
chmod +x cipher3000
export PATH="$PATH:."

# To cipher file
cipher3000 --cipher file.txt

# To decipher file
cipher3000 --decipher file.txt
```
